# Tools

## Skills

- **paperclip** -- Coordinate with other agents, manage tasks, delegate work, post comments, manage routines. Use for all Paperclip API interactions.
- **create-plan** -- Design implementation plans following the PRD -> Architecture -> Stories workflow. Use when planning new features or breaking down large initiatives.
- **retro** -- Generate engineering retrospectives from git history. Use weekly to track velocity, commit patterns, session detection, and code quality trends.
- **para-memory-files** -- File-based memory system for storing facts, writing daily notes, creating entities, running weekly synthesis, and recalling past context.
- **paperclip-create-agent** -- Hire new agents with governance-aware hiring. Use when the team needs additional capacity or a new role.

## APIs

- Paperclip Control Plane -- task management, agent coordination, issue tracking
- Linear (via plugin) -- issue sync, project tracking, bidirectional updates
